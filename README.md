# 名刺フォローアップAI (CardApp)

## 概要
名刺を撮影してAI分析し、お礼メールを自動生成するアプリケーションです。

## セットアップ

### 必要要件
- Python 3.x
- VS Code (推奨)

### インストール
依存ライブラリをインストールします。
```bash
pip install -r requirements.txt
```

## 実行・デバッグ方法

### VS Codeを使用する場合 (推奨)
1. VS Codeでこのフォルダを開きます。
2. 左側の「実行とデバッグ」アイコン (または `Ctrl+Shift+D`) をクリックします。
3. 上部のプルダウンから「Python: Flask」を選択します。
4. 緑色の再生ボタン (または `F5` キー) を押してデバッグを開始します。
5. ブラウザで http://127.0.0.1:5000 にアクセスします。

ブレークポイントを設定することで、コードの任意の場所で処理を一時停止し、変数の状態などを確認できます。

### コマンドラインから実行する場合
```bash
python app.py
```
